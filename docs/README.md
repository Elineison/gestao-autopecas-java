# Documentação do Trabalho

Esta pasta reúne os materiais usados na entrega da disciplina Imersão Profissional: Implementação de uma Aplicação.

## Arquivos Principais

- `PAPER.md`: texto base do paper, com resumo, introdução, fundamentação teórica, metodologia, considerações e referências.
- `PPT_SOCIALIZACAO.md`: roteiro para montar os slides da socialização.
- `ETAPA_PASSOS_1_2.md`: descrição da necessidade da aplicação e da implementação.

## Apoio Para Finalizar

- `GUIA_ENTREGA_FINAL.md`: checklist do que falta fazer antes de enviar.
- `PRINTS_E_EVIDENCIAS.md`: lista dos prints e trechos de código que devem entrar no paper e no PPT.
- `ROTEIRO_APRESENTACAO_ORAL.md`: fala sugerida para apresentar o trabalho de forma natural.

## Modelos Oficiais

Os modelos oficiais copiados para a raiz do projeto são:

- `modelo-paper-uniasselvi.docx`;
- `template-socializacao-uniasselvi.pptx`.

O conteúdo dos arquivos `.md` deve ser usado como base para preencher esses modelos.
